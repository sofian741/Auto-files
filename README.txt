File Organizer GUI
------------------

A simple Windows tool with a graphical interface that automatically sorts files in any folder you choose.

Features:
---------

Easy-to-use GUI â€“ no CMD needed

Organizes files into subfolders:

DLL

Rar (RAR / ZIP)

EXE

ISO

IMAGE

Else (other types)

Ignores folders and .py files

Skips files currently in use

Does not move the program itself

Shows a message when finished, including skipped files

How to Use:
-----------

Run the .exe file.

Click Select Folder & Organize.

Choose the folder you want to organize.

The program will sort files automatically.

No installation required.

Requirements:
-------------

Windows OS


No Python installation needed when using the compiled .exe
